# trampos-cli

- procura trampos de programação no trampos.co

# Requirements

- Node.js 
  - for dev: ^8.16.0

# Instalação

- Vá até a pasta do projeto
- `npm install -g`

# Como usar

- `trampos-cli`

# Opções

```
Usage: trampos-cli [options]

Options:
  -s, --skill <name>  specify the programming language or software
  -h, --help          output usage information
```
